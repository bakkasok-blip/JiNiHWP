# JiNi HWP 개인정보처리방침 (Privacy Policy)

최종 수정일: 2026-07-13

## 개요

JiNi HWP("본 확장프로그램")는 한글(HWP/HWPX) 문서를 브라우저에서 열람·편집·저장하는
크롬 확장프로그램입니다.

## 데이터 수집

**본 확장프로그램은 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.**

- 사용자가 여는 문서 파일은 **사용자의 브라우저 안에서만(WebAssembly)** 처리되며,
  개발자 또는 제3자의 서버로 전송되지 않습니다.
- 방문 기록, 검색어, 입력 내용 등 어떠한 이용 데이터도 수집하지 않습니다.
- 분석 도구(애널리틱스), 광고, 추적 기술을 사용하지 않습니다.

## 저장되는 정보

다음 정보만 브라우저의 확장프로그램 저장소(chrome.storage)에 **로컬로** 저장됩니다.

- 확장프로그램 설정값 (다운로드 자동 열기 여부, 링크 배지 표시 여부 등)

이 정보는 기기 밖으로 전송되지 않으며, 확장프로그램 삭제 시 함께 삭제됩니다.

## 권한 사용 목적

| 권한 | 목적 |
|------|------|
| downloads | HWP/HWPX 파일 다운로드를 감지해 뷰어에서 자동으로 열기 위해 사용 |
| contextMenus | HWP 링크의 우클릭 메뉴("JiNi HWP로 열기")를 제공하기 위해 사용 |
| storage | 사용자 설정을 기기에 로컬 저장하기 위해 사용 |
| clipboardWrite | 편집기의 복사(Ctrl+C) 기능을 위해 사용 |
| activeTab | 현재 탭의 HWP 링크를 인식하기 위해 사용 |
| 호스트 권한(<all_urls>) | 사용자가 요청한 HWP 파일을 해당 사이트에서 내려받아 표시하고, 웹페이지의 HWP 링크 옆에 배지를 표시하기 위해 사용 |
